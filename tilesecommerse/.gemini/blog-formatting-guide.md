## How to Use the Blog Editor

### Method 1: Using the Toolbar (Recommended)

When creating blog content, use the rich text editor toolbar to format your content:

1. **Bold Text**: Select text and click the **B** button
2. **Italic Text**: Select text and click the *I* button
3. **Headings**: Click H1, H2, or H3 buttons for different heading levels
4. **Lists**: 
   - Click the bullet list icon for unordered lists
   - Click the numbered list icon for ordered lists
5. **Line Breaks**: Press `Shift + Enter` for a line break within a paragraph
6. **New Paragraph**: Press `Enter` for a new paragraph

### Method 2: Pasting from ChatGPT or Other Sources

When you paste formatted text from ChatGPT or other sources:

1. **Paste the text** directly into the editor
2. **Select the pasted content**
3. **Apply formatting** using the toolbar:
   - For lists: Select the list items and click the bullet or numbered list button
   - For bold text: Select the text and click the Bold button
   - For headings: Select the heading text and click H1, H2, or H3

### Example: Creating a List

If you have this text from ChatGPT:
```
1. Understand the Area

Different areas need different types of tiles:

Living room: Go for vitrified or porcelain tiles for a premium look.
Bathroom: Choose anti-skid tiles for safety.
Kitchen: Matte or textured tiles help prevent slipping and stains.
Outdoor areas: Use heavy-duty, weather-resistant tiles.
```

**Steps to format it:**

1. Type or paste: "1. Understand the Area"
2. Select it and click **H2** or **H3** to make it a heading
3. Press Enter to create a new paragraph
4. Type: "Different areas need different types of tiles:"
5. Press Enter
6. Click the **bullet list** button (or numbered list)
7. Type each item:
   - Living room: Go for vitrified or porcelain tiles for a premium look.
   - Bathroom: Choose anti-skid tiles for safety.
   - Kitchen: Matte or textured tiles help prevent slipping and stains.
   - Outdoor areas: Use heavy-duty, weather-resistant tiles.
8. For bold text within list items, select the text (e.g., "Living room:") and click **B**

### Preview Your Content

Always click the **Preview** button before saving to see exactly how your blog will look when published. The preview now uses the same styling as the published blog.

## Formatting Tips

### Line Breaks and Spacing
- **Single line break**: Press `Shift + Enter`
- **New paragraph** (with spacing): Press `Enter`
- The editor automatically adds proper spacing between paragraphs and list items

### Lists
- **Bullet lists**: Best for unordered items
- **Numbered lists**: Best for steps or sequential items
- You can nest lists by pressing `Tab` while in a list item

### Headings
- **H1**: Main title (rarely used in content, as the blog title is H1)
- **H2**: Major sections
- **H3**: Subsections

### Bold vs Strong Emphasis
- Use **bold** for important terms or key points
- Don't overuse bold - it loses impact

### Links
1. Select the text you want to link
2. Click the link icon
3. Enter the URL
4. The link will appear in orange

### Images
1. Click the image icon
2. Enter the image URL
3. The image will be inserted and automatically styled
